# Lumis SDK 0.0.7

Lumis SDK 0.0.7 completes Sprint 3 with durable shared-memory contracts and a
vendor-neutral connector proof.

## Highlights

- Accepted RFC 0002 for provider-neutral memory configuration, secret references,
  migrations, idempotency, bounded retrieval, and authority boundaries.
- Added reusable asynchronous `MemoryStore` contracts and an async SQLite reference adapter.
- Added strict PostgreSQL memory configuration for independently packaged adapters.
- Added the independent `lumis-sdk-postgres-memory` plugin and a two-process Docker cookbook.
- Added authenticated webhook normalization with timestamp and replay protection.
- Added strict HTTP JSON evidence-provider configuration.
- Added the independent `lumis-sdk-http-json-evidence` plugin and an offline fixture cookbook.
- Added dedicated PostgreSQL and HTTP connector contract jobs to CI.

## Safety and compatibility

SQLite remains the default and core installs no PostgreSQL or HTTP client dependency. Database
credentials, webhook secrets, and connector tokens are referenced through environment variables.
Memory operations, payloads, retries, responses, and retrieval are bounded. Installing a plugin
does not grant network, secret, filesystem, model, or execution authority automatically.

The optional plugin sources are maintained independently under `packages/`; this core release
provides their required public SDK contracts but does not silently install either plugin.

## Install

```bash
uv add "lumis-sdk==0.0.7"
```

or:

```bash
pip install "lumis-sdk==0.0.7"
```
