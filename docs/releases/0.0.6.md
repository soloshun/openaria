# Lumis SDK 0.0.6

Lumis SDK 0.0.6 completes the metadata-first plugin SDK foundation.

## Highlights

- Accepted RFC 0001 for plugin discovery, compatibility, capability negotiation, and loading.
- Strict versioned `PluginManifest` with checked JSON Schema.
- Metadata-only `lumis plugins list` and `lumis plugins doctor`.
- Lazy `importlib.metadata` discovery that does not import plugin modules.
- Isolated missing, invalid, incompatible, archived, duplicate, import, and factory failures.
- Default-denied network, filesystem, secret, model, and execution authorities.
- Reusable plugin manifest/factory fixtures and contract assertions.
- Independently packaged synthetic plugin built and installed beside the SDK in a clean environment.

## Safety and compatibility

Core operation remains unchanged when no plugins are installed. Installing or discovering a
plugin grants no runtime authority. Static manifests are bounded to 64 KiB, and third-party
Python is imported only after an application explicitly selects and loads a compatible plugin.

## Install

```bash
uv add "lumis-sdk==0.0.6"
```

or:

```bash
pip install "lumis-sdk==0.0.6"
```
