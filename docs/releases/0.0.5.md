# Lumis SDK 0.0.5

Lumis SDK 0.0.5 adds explicit bounded comparisons over list-valued structured fields, resolving
issue #25.

## Added

- `anyElement` applies one existing scalar operator and passes when at least one element matches.
- `allElements` applies one existing scalar operator and passes when every element matches.
- Condition explanations include matched element indexes and bounded observed values.
- A stable list-valued testkit fixture and updated structured-rule cookbook.

## Safety and compatibility

Existing scalar conditions are unchanged. Lists are never implicitly converted to strings.
Empty lists fail both quantifiers. Non-list values, nested collections, and lists above 100
elements fail closed.

## Install

```bash
uv add "lumis-sdk==0.0.5"
```

or:

```bash
pip install "lumis-sdk==0.0.5"
```
