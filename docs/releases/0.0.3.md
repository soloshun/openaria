# Lumis SDK 0.0.3

Lumis SDK 0.0.3 completes the v0.2 evidence and machine-readable reporting foundation.

## Highlights

- Vendor-neutral asynchronous `EvidenceProvider` port.
- Bounded evidence collection with timeouts, filtering, duplicate handling, truncation, and
  conservative redaction.
- Offline local JSON evidence adapter with structured failures.
- Strict versioned JSON diagnosis reports and checked JSON Schema.
- Reusable evidence/report fixtures, fakes, and contract assertions.
- CLI and doctor composition for local evidence and JSON reports.
- Synthetic evidence-to-report cookbook and public Python API documentation.

## Compatibility and safety

Markdown remains the default report format, and projects without `evidenceProviders` retain their
existing behavior. Evidence and JSON reporting are opt-in.

Evidence is treated as untrusted input. The release adds no model requirement, hosted dependency,
telemetry, or action-execution authority.

## Install

```bash
uv add "lumis-sdk==0.0.3"
```

or:

```bash
pip install "lumis-sdk==0.0.3"
```
